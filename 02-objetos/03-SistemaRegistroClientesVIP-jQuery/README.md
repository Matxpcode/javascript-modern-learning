# Sistema de Registro de Clientes VIP

## Descripción

Aplicación desarrollada con **JavaScript**, **jQuery** y **Bootstrap** que registra la información de un cliente, evalúa si cumple las condiciones para ser un cliente **VIP Oro** y muestra el resultado en una tarjeta con el resumen de la evaluación.

## Objetivos del ejercicio

- Practicar la creación y manipulación de objetos.
- Capturar eventos utilizando **jQuery**.
- Obtener valores de formularios con `.val()` y `.prop()`.
- Modificar dinámicamente el DOM con `.text()`, `.html()`, `.addClass()` y `.removeClass()`.
- Aplicar reglas de negocio mediante estructuras condicionales.

---

## Tecnologías utilizadas

- HTML5
- Bootstrap 5
- JavaScript (ES6)
- jQuery 3.7

---

## Conceptos aplicados

### JavaScript

- Objetos
- Variables

- Numbers

- Selección de elementos (`$("#id")`)

---

## Funcionamiento

1. El usuario registra sus datos.
2. Se crea un objeto `cliente`.
3. Se evalúan las condiciones para determinar si es un cliente **VIP Oro**.
4. Se calcula el descuento correspondiente.
5. Se muestran los resultados en una tarjeta.
