# JavaScript Completo

Repositorio de estudio y práctica de **JavaScript**, organizado por bloques temáticos. Cada carpeta contiene teoría, ejercicios y proyectos desarrollados durante el aprendizaje del lenguaje.

---

## 🛠 Tecnologías

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6+)
- jQuery *(cuando el tema lo requiera)*

---

## 📂 Organización del repositorio

Cada bloque del curso mantiene la misma estructura para facilitar la navegación.

```text
📁 Bloque del curso
│
├── 📁 01-Ejercicio
│   ├── index.html
│   ├── app.js
│   ├── style.css (si aplica)
│   └── README.md  (img)
│
├── 📁 02-Ejercicio
│
├── 📁 03-Ejercicio-jQuery (si aplica)
│
├── 📁 D01-Desafio
│
├── 📁 D02-Desafio
│
└── 📁 D03-Desafio-jQuery (si aplica)
```

Cada ejercicio incluye un **README** con la descripción del problema, conceptos aplicados y tecnologías utilizadas.
