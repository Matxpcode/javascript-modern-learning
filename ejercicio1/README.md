# Ejercicio 1 - Calculadora de descuentos

## Objetivo

Construir un formulario que permita calcular descuentos según el tipo de cliente.

## Conceptos aplicados

- Variables
- Eventos
- DOM
- Condicionales
- parseFloat()
- trim()
- Manipulación de clases
- Validación de formularios

## Tecnologías

- HTML
- Bootstrap
- JavaScript

## Lo aprendido

- Capturar eventos submit.
- Validar entradas.
- Manipular el DOM.
- Mostrar y ocultar componentes.
