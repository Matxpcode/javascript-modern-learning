# Inventario de Productos

## Descripción

Aplicación desarrollada en JavaScript para registrar productos dentro de un inventario utilizando un arreglo de objetos y mostrar información resumida del registro.

## Funcionalidades

- Registrar productos desde un formulario.
- Almacenar cada producto como un objeto dentro de un array.
- Mostrar el primer y último producto registrado.
- Mostrar el total de productos almacenados.
- Actualizar dinámicamente el contador de registros.
