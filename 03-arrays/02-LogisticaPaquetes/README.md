# Gestion Paquetes

## Caso práctico

Desarrollar un sistema que permita registrar múltiples paquetes de una empresa de logística utilizando **Arrays de Objetos**. Cada paquete debe almacenarse dentro de una colección y posteriormente mostrar información resumida sobre los registros realizados.

## Requerimientos

- Registrar un paquete mediante un formulario.
- Almacenar cada paquete como un objeto dentro de un Array.
- Registrar múltiples paquetes durante la ejecución.
- Mostrar la cantidad total de paquetes registrados.
- Mostrar la información del primer paquete registrado.
- Mostrar la información del último paquete registrado.
- Utilizar una función reutilizable para mostrar la información de un paquete.
